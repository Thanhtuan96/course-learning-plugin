           - Timestamp: `[[YYYY-MM-DD]]` (e.g., [[2026-03-06]])
           - Type tag: `[Question]`, `[Idea]`, `[Insight]`, or `[To explore]`
           - Format:
             ```
             [[2026-03-06]] [Question]
             Your note content here...
             ```
       
       d) Append to NOTES.md:
          - Read existing NOTES.md (or create if doesn't exist)
          - Prepend new entry (newest at top - feed style)
          - Write back to file
       
       e) Show confirmation:
          - Display saved note with timestamp and type
           - Show "Saved to: courses/{slug}/NOTES.md"
           - Mention: "Add another note? Run `/professor:note` anytime."
