---

## How to Use These Templates

1. **Start with tutorials** if the user needs foundational understanding
2. **Use official docs** when they need precise reference material
3. **Conference talks** work well for high-level overview and expert perspectives
4. **Stack Overflow** helps with specific error messages and common pitfalls

**Remember:** Present these as starting points for the user's own research journey. Frame as "here are some directions to explore" not "go figure it out yourself."
